# BIS-TAGSET-OF-MAGAHI-
MAGAHI IS LOW RESOURCES LANGAUGE 
Magahi is a low-resource Indian language with limited NLP tools and datasets. Developing NER systems for such languages is challenging due to lack of annotated corpora and training data.

This work focuses on:

Applying NER to Magahi text
Using BIS standard POS tagset (33 tags)
Handling linguistic challenges using rule-based method
